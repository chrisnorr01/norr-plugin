# Skill: norr-plan

Erstellt einen Content-Kalender fuer Norr ueber einen definierten Zeitraum.

## Was geplant werden kann

- Instagram (@chris.norrweg) — Posts, Reels, Stories
- LinkedIn — Beitraege
- Pinterest — Pins und Boards
- YouTube Shorts
- Kombiniert: welches Thema passt auf welche Plattform

## Pflichtregeln beim Planen

**Fuer jeden geplanten Post:**
- Datum / Woche
- Plattform
- Thema (ein Satz)
- Format (Foto / Video / Karussell / Short)
- Status: Idee / In Arbeit / Fertig

**Rhythmus-Empfehlung:**
- Instagram: 3-4x pro Woche
- LinkedIn: 1-2x pro Woche
- Pinterest: taeglich (koennen aeltere Posts wiederverwendet werden)
- YouTube Shorts: 1-2x pro Woche

**Themenquellen:**
- Aktuelle Projekte aus dem Vault
- Saisonale Themen (Fjällräven Classic, Winter, Fruehling im Fjäll)
- Gear-Themen aus dem Fachhandel
- Natur-Fakten und Oekosystem-Wissen

## Vorgehen

1. Zeitraum fragen (z.B. naechste 2 Wochen)
2. Was gerade aktuell ist fragen (Projekte, Events, Themen)
3. Kalender als Tabelle erstellen
4. Auf Wunsch direkt einzelne Posts ausschreiben
