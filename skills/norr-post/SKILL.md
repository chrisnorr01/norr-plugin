# Skill: norr-post

Erstelle einen Instagram-Post für Chris (@chris.norrweg) basierend auf dem angegebenen Thema.

## Pflichtregeln

**Aufbau:**
1. **Opener** — ein einziger kurzer Satz. Behauptung oder Wissensluecke. Keine Begruessung.
2. **Begruessung** — "Hej" wenn nordisch/schwedisch, "Moin" wenn bodestaendig/kuestennah
3. **Fliesstext** — 3-5 kurze Absaetze. Lehrend aber nicht belehrend. Persoenlich, direkt.
4. **Abschluss** — eine echte Frage an die Community. Kein "Folgt mir".

**Stimme:**
- Du-Ansprache immer
- Kurze Saetze die atmen
- Nordische Begriffe erlaubt: Friluftsliv, Jedermansrecht, Allemansraetten
- LNT-Haltung mitdenken

**Verboten:**
- "atembeaubend", "wunderschoen", "unvergesslich", "lohnenswert"
- Bindestriche als Gedankenstriche
- Hashtags im Fliesstext
- Engagement-Bait

**Am Ende:** 5-8 Hashtags, darunter immer #norr #friluftsliv

## Zielgruppe

Lea, ~32, outdoor-aktiv, nordisch interessiert, echte Erfahrungen statt Hochglanz.

## Vorgehen

1. Thema entgegennehmen
2. Bei Bedarf recherchieren
3. Post schreiben und vorlegen
4. Anpassungen vornehmen
