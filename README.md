# norr-plugin

Persoenliches ECC-Plugin fuer Norr — Content, Recherche und Planung.

## Skills

| Command | Was es tut |
|---------|-----------|
| `/norr:norr-post` | Instagram-Post im Norr-Stil schreiben |
| `/norr:norr-linkedin` | LinkedIn-Beitrag erstellen |
| `/norr:norr-pinterest` | Pinterest-Pin-Texte schreiben |
| `/norr:norr-shorts` | YouTube Shorts Skript erstellen |
| `/norr:norr-buch` | Buchpassage im Ransmayr-Stil |
| `/norr:norr-plan` | Content-Kalender planen |
| `/norr:norr-recherche` | Outdoor/Textil/Gear recherchieren |
| `/norr:gear-check` | Ausruestung analysieren und bewerten |
| `/norr:natur-bio` | Biologie der Natur erklaert |
| `/norr:natur-chemie` | Chemische Prozesse in der Natur |

## Agents

- `norr-writer` — Schreibt alle Norr-Inhalte in Chris' Stimme
- `norr-researcher` — Recherchiert Outdoor, Natur, Gear mit Tiefe

## Installation

```
claude plugins install chrisnorr01/norr-plugin
```
